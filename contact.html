聯絡 PokeSwipe

PokeSwipe 由獨立開發者維護，沒有客服中心。所有溝通都透過 GitHub 公開處理，效率最高、也最透明。

📮 主要聯絡管道

🐛 Bug 回報 / 功能建議

請開 GitHub Issue（最推薦的方式）：

➜ 開啟新 Issue

Tip: 附上截圖、瀏覽器版本（Chrome/Safari/Firefox）、作業系統（iOS/Android/Mac/Windows），會更快被處理。

🗑 刪除上傳請求

想從伺服器永久刪除你上傳的截圖？請提供：

  • 圖片的 public_id（在「我的上傳」按右鍵可看到）
  • 或截圖中的 12 位好友碼

➜ 提交刪除請求

隱私 Tip: 如果不想公開，可以在 Issue 內留下隨機 ID，私訊發 GitHub 帳號補資料。

©️ 版權與 DMCA 通報

如果發現有人盜用你的截圖或侵犯你的版權，請提供：

  • 被侵權的圖片 URL 或 public_id
  • 你的版權證明（原始檔、時間戳等）
  • 聯絡 Email（會在處理完後告知結果）

➜ 提交 DMCA 通報

🚨 不當內容檢舉

看到不當截圖（個資、廣告、釣魚連結等）？請提供圖片 ID 或截圖描述：

➜ 檢舉內容

🤝 合作 / 媒體採訪

想合作、做訪談、轉載分享？歡迎開 GitHub Discussion 或 Issue 標記為 partnership。

➜ 開啟 Discussion

⏱ 回覆時間

  • 緊急刪除 / 隱私問題：24-48 小時內
  • Bug 回報：3-7 天內回覆，依嚴重程度排優先級
  • 功能建議：不一定立即回，但會被列入 roadmap 評估

本站由獨立開發者業餘維護，請耐心等候。複雜問題的修正可能需要數週時間。

❓ 在聯絡前，建議先看

  • 常見問題 FAQ——大部分問題在這裡都有答案
  • 關於 PokeSwipe——了解技術架構與設計理念
  • 隱私政策——資料處理相關說明
  • 使用條款——服務範圍與責任
