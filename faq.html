遇到問題？這裡列出最常被問到的 20+ 個問題。

🎯 基本問題

PokeSwipe 是什麼？

PokeSwipe 是一個專為 Pokémon GO 玩家設計的好友碼分享平台。 用「滑卡片」的方式瀏覽其他玩家的訓練家代碼截圖，看到喜歡的就一鍵複製、回到遊戲加好友。

需要付費嗎？需要註冊嗎？

完全免費、不需要註冊。沒有任何帳號系統，沒有 Email 驗證，沒有付費方案。

支援哪些裝置和瀏覽器？

所有現代瀏覽器都支援：

  • iPhone：Safari 14 以上
  • Android：Chrome / Edge / Samsung Internet 最新版
  • 桌面：Chrome、Firefox、Safari、Edge 最新版

建議使用直向手機螢幕獲得最佳體驗（卡片會更大）。

為什麼網站要叫 "PokeSwipe"？

"Poke" 來自 Pokémon，"Swipe" 是滑動的意思——靈感來自交友軟體的滑卡片設計， 把找寶可夢朋友變成像滑 Tinder 一樣輕鬆。

📤 上傳問題

怎麼正確截圖好友碼？
  1. 打開 Pokémon GO
  2. 點左下角的訓練家頭像
  3. 點右上角的 QR Code 圖示，進入訓練家代碼頁面
  4. 手機直向截圖（必須是直向！）
  5. 確認截圖中清楚可見：你的暱稱、12 位好友碼、QR Code
  6. 回到 PokeSwipe，切到「上傳截圖」分頁，選擇剛剛的截圖
為什麼上傳失敗，顯示「找不到好友碼」？

可能原因：

  • 截圖不是訓練家代碼頁面（必須是顯示 12 位數字 + QR Code 的那一頁）
  • 圖片過暗、模糊、或被裁切
  • 不是 Pokémon GO 的截圖
  • 不是直向（橫向截圖會被拒絕）

請重新截一張清楚的直向截圖再試。

顯示「此好友碼已上傳過」？

系統會自動偵測重複——同一個 12 位好友碼只能被上傳一次。如果是你之前上傳的， 可以從「我的上傳」找到並刪除舊的，再上傳新的。

檔案大小有限制嗎？

每張截圖最大 15 MB，支援 JPG、PNG、HEIC（iPhone 預設格式）。 手機原始截圖通常都在 1-3 MB 之間，遠小於上限。

每天可以上傳幾張？

每個裝置每天 5 張，凌晨 0 點重置。這個限制是為了避免少數人洗版，讓大家都有公平的曝光機會。

可以一次上傳多張嗎？

目前只支援單張上傳。要上傳第二張請等第一張完成後再選擇下一張。

🔍 滑動瀏覽問題

怎麼複製好友碼？

手機：點卡片下方的大按鈕「一鍵複製好友碼」，複製完切到 Pokémon GO → 好友 → 新增好友 → 貼上。

桌面：按 C 快捷鍵，或點圖示按鈕。

桌面有哪些鍵盤快捷鍵？
  • ← / →：下一張卡片
  • C：複製當前好友碼
  • Space：放大查看 QR Code（lightbox）
  • Esc：關閉 lightbox 或 help
看完所有卡片後會怎樣？

會自動從頭開始，重新顯示所有人的卡片，並提示「已看完全部，重頭開始」。 系統會記住你看過哪些，下次回來會從上次停的地方繼續。

有新的好友碼上傳會自動更新嗎？

不會即時更新。為了減少 API 呼叫量，列表會在以下時機重新整理：

  • 重新載入頁面
  • 切換到「上傳截圖」再切回來
  • 每 5 分鐘自動更新一次

🛡️ 隱私與安全

會收集我的個人資料嗎？

不會。具體說明：

  • 不收集：Email、姓名、IP 位址、地理位置、瀏覽器指紋、社交帳號
  • 有儲存：你上傳的截圖（公開）、隨機產生的 Trainer ID（只在你的瀏覽器 localStorage）

詳見 隱私政策。

截圖會被誰看到？

所有訪客都能看到。這就是這個平台的目的——讓世界各地的訓練家看到你的好友碼。 不要在截圖裡顯示任何遊戲外的個人資訊。

使用 cookies 嗎？

PokeSwipe 本身不使用 cookies，只使用 localStorage（存在你的瀏覽器，不會傳到任何伺服器）。 第三方服務（Cloudinary、Google Fonts）可能設定自己的 cookies。

怎麼從伺服器永久刪除我的截圖？

「我的上傳」清單裡的「刪除」按鈕只會在你的瀏覽器隱藏該圖。如需從 Cloudinary 永久刪除， 請透過 GitHub Issues 或 聯絡頁提供 public_id，我們會盡快處理。

小孩也能用嗎？

本站不針對 13 歲以下兒童設計。雖然不收集個資、無聊天功能，但仍建議家長陪同使用。

💡 進階技巧

怎麼在卡片上看出哪些是新的？哪些是我看過的？

左上角的卡片角標會顯示：

  • 白點：尚未看過
  • 淡綠色點：已看過
  • 橘色點：你自己上傳的
OCR 識別錯了，可以重新掃描嗎？

可以。點卡片放大進入 lightbox，按「重新掃描」按鈕，系統會用更高解析度再跑一次 OCR。

可以把 PokeSwipe 加到主畫面嗎？

可以！PokeSwipe 是 PWA：

  • iPhone：Safari 點分享 → 加入主畫面
  • Android：Chrome 點選單 → 加入主畫面 / 安裝應用程式
